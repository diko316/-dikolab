# 0.1.0 — Initial Release

**Date:** 2026-04-20

## Summary

First release of `@dikolab/tokenizer`, migrated
from
[`libcore-tokenizer`](https://github.com/nicoseta/libcore-tokenizer)
with full TypeScript support.

## What's included

- `Tokenizer` class with `define()` and
  `tokenize()` methods
- NFA-based regex-to-state-machine compilation
- State machine serialization via `toJSON()`,
  `toObject()`, and `fromJSON()`
- Supported regex operators: alternation,
  optional, repeat, Kleene star, character
  classes, negated classes, ranges, grouping
