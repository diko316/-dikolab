# 0.1.0 — Initial Release

**Date:** 2026-04-20

## Summary

First release of `@dikolab/lalr-parser`, migrated
from
[`libcore-parser-lalr`](https://github.com/nicoseta/libcore-parser-lalr)
with full TypeScript support.

## What's included

- `Parser` class with `define()`, `parse()`,
  `iterator()`, `toJSON()`, `toObject()`, and
  `fromJSON()` methods
- `define()` function for grammar definition
  from alternating name/production arrays
- `load()` function for restoring parsers from
  serialized JSON
- `BaseIterator` / `Iterator` pushdown automaton
  with shift-reduce state machine
- `Lexeme` parse tree node with parent/child/
  sibling links
- `registerIterator()` for custom iterator
  implementations
- `debug()` toggle for parser diagnostics
- `isParser()` type guard
- LALR(1) DFA state generation via closure
  computation
- Grammar state serialization and import/export
