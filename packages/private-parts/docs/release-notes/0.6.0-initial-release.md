# 0.6.0 — Initial Release

**Date:** 2025-09-21

## Summary

First published release of `@dikolab/private-parts`.

Provides WeakMap-based private data storage accessible
by object instance and property key. Designed for
encapsulating internal state in class hierarchies
without exposing private fields.

## Features

- `get(instance, key)` — retrieve a private value
- `set(instance, key, value)` — store a private value
- `clear(instance)` — remove all private values for
  an instance
- `clearAll()` — reset all private storage (useful
  for test teardown)
- ESM, CJS, UMD, and raw TypeScript (Deno) outputs
- Full TypeScript type declarations
