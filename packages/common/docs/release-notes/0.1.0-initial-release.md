# 0.1.0 — Initial Release

**Date:** 2026-04-20

## Summary

First release of `@dikolab/common`, migrated from
[`libcore`](https://github.com/nicoseta/libcore)
with full TypeScript support.

## What's included

- Type checking: `isString`, `isNumber`,
  `isObject`, `isArray`, `isMethod`, `isRegex`,
  `isDate`, `isScalar`, `isThenable`,
  `isIterable`, `isNativeObject`, `type`,
  `signature`
- Object utilities: `each`, `assign`, `contains`,
  `clone`, `compare`, `fillin`, `clear`, `rehash`,
  `instantiate`
- String utilities: `camelize`, `uncamelize`,
  `encode64`, `decode64`, `utf2bin`, `bin2utf`,
  `trim`
- JSON path: `jsonFind`, `jsonSet`, `jsonExists`,
  `jsonClone`, `jsonEach`, `jsonCompare`,
  `jsonUnset`, `jsonFill`, `jsonParsePath`
- Array set operations: `unionList`,
  `intersectList`, `differenceList`
- Environment detection: `browser`, `nodejs`
- Middleware: `run`, `register`, `middleware`,
  `Registry`, `createRegistry`
- Async: `setAsync`, `clearAsync`
