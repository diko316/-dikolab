# 0.1.0 — Infrastructure Overhaul

**Date:** 2026-04-19

## Summary

Major infrastructure upgrade with no API changes.
Migrated build system from Rollup to esbuild,
upgraded ESLint to strict type-checked presets, and
added publishing etiquette files.

## Changes

- Migrated from Rollup to esbuild with quad output
  (CJS, ESM, UMD, Deno raw TypeScript)
- Upgraded ESLint from `recommended` to
  `strictTypeChecked` + `stylistic` presets
- Enabled type-aware linting with `projectService`
- Added `consistent-type-imports` and
  `consistent-type-exports` rules for esbuild
  tree-shaking
- Aligned Prettier with documented code style
  (3-space indent, 72-char width, LF)
- Added ISC LICENSE file to package
- Added installation instructions and badges to
  README
- Fixed template expression type safety issues
- Removed unnecessary generic type parameters
- Added eslint-disable for intentional `any` returns
  from `@dikolab/private-parts` storage API

## Breaking Changes

None. Public API remains unchanged.
