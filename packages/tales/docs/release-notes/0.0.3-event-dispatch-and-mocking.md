# 0.0.3 — Event Dispatch and Mocking

**Date:** 2025-09-21

## Summary

Added event dispatching for core lifecycle actions
and use-case mocking for testing support.

## Features

- Dispatch events on `assume` (Actor), `perform`
  (Usecase), and `achieved` (Goal) actions
- `listen()` function to subscribe to use-case and
  goal events
- Use-case handler mocking for unit testing
- Declare scopes, goals, use-cases, actors, and roles
- Execute use-cases via `assume(actor).perform()`
- ESM, CJS, UMD, and raw TypeScript (Deno) outputs
- Full TypeScript type declarations
